# Visualizador de Perfil do GitHub

Este é um projeto simples de visualização de perfil do GitHub, desenvolvido para demonstrar a busca de usuários e a exibição de informações básicas do perfil.

## Funcionalidades

- **Buscar Usuário:** Permite buscar um perfil do GitHub digitando o nome de usuário em um campo de entrada.
- **Acionamento da Busca:** A busca pode ser acionada de duas formas:
    - Clicando no botão "Buscar".
    - Pressionando a tecla "Enter" no campo de entrada.

## Como Usar

1.  Clone este repositório para sua máquina local.
2.  Abra o arquivo `index.html` em seu navegador web.
3.  Digite um nome de usuário do GitHub no campo de busca e pressione "Enter" ou clique no botão "Buscar".
4.  (Em desenvolvimento) As informações do perfil serão exibidas na seção de resultados.

## Tecnologias Utilizadas

-   HTML5
-   CSS3
-   JavaScript
